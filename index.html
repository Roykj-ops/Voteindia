🔥 Firebase Setup Guide

Step 1: console.firebase.google.com pe jaao

Step 2: "Add project" → naam dalo → Continue → Create

Step 3: Left menu: Authentication → Get Started → Phone → Enable karo → Save

Step 4: Left menu: Firestore Database → Create database → Test mode → Next → Done

Step 5: ⚙️ Project Settings (gear icon) → General tab → "Your apps" → Web icon (</>) → App naam dalo → Register

Step 6: Config copy karo aur is file mein firebaseConfig object update karo

Step 7: Authentication → Settings → Authorized domains mein apna domain add karo (local ke liye localhost already hota hai)
